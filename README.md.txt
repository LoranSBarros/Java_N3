Sistema de Gerenciamento de Estoque de Concessionária de Motocicletas

Este projeto implementa um sistema para gerenciar o estoque de uma concessionária de motocicletas, com foco em registrar vendas, calcular lucros e gerenciar o estoque de forma eficiente.

Objetivo

O sistema foi desenvolvido para facilitar o gerenciamento de vendas e reposição de motocicletas em uma concessionária. Ele permite:
- Adicionar motocicletas ao estoque.
- Registrar as vendas realizadas.
- Calcular o lucro de cada venda.
- Monitorar o estoque de motocicletas.

Funcionalidades

- Adicionar motocicletas ao estoque: O usuário pode adicionar novas motocicletas ao estoque, informando o modelo, preço de compra e preço de venda.
- Registrar vendas: As vendas realizadas são registradas, incluindo o modelo da motocicleta e a quantidade vendida.
- Calcular lucro: O sistema calcula o lucro gerado por cada venda, subtraindo o preço de venda do preço de compra.
- Listar estoque: O estoque de motocicletas pode ser listado, mostrando as motocicletas disponíveis.


Como Executar

1.Abra o projeto no IntelliJ IDEA.
2.Compile o código
3.Execute o código:
   - Clique com o botão direito na classe `Main.java` e selecione Run Main.main().
